INFLUX_DB SHELL

> SHOW SERIES ON DATA_BASE_NAME - show all the keys of fields (if previusly wasn't used 'USE DATABASE_NAME')

> SHOW DATABASES - show all DB

> SHOW MEASUREMENTS ON DB_NAME - show ameasurements if table, equivalent to table in MySQL

> SHOW SERIES ON DB_NAME - each and every record of measurements (it is like roww of the table)

> SHOW TAG VALUES ON DB_NAME - Returns the list of tag values for the specified tag key(s) in the database.

> SHOW TAG KEYS ON DB_NAME - Optional column of the table. It is indexed column for faster search

> SHOW FIELD JEYS ON DB_NAME -  This is the column which stores the actual data you want to measure. Like response time of a page.
